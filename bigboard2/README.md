# Big Board II

The **Big Board II** (Digital Research Computers, 1982) is a Z80 CP/M
single-board computer — the successor to the original Big Board, and a relative
of the Xerox 820 line.

MAME system `bigbord2`, driver
[`xerox/bigbord2.cpp`](https://github.com/mamedev/mame/blob/master/src/mame/xerox/bigbord2.cpp).
PR [mamedev/mame#15412](https://github.com/mamedev/mame/pull/15412) boots CP/M,
wires the Xebec/SASI hard disk, and marks the system working (it fixes a
DMA-RDY multiplexer bug that previously stopped it booting).

> This folder is just getting started — for now it holds a photo. Boot disks
> and ROM notes will follow.
