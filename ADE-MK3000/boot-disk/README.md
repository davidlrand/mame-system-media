# MK3000 boot disk — reconstruction in progress

No original MK3000 media is known to survive. A bootable CP/M 2.2 disk is
being reconstructed from the BIOS disassembly listing (`../docs/`) and the
disk-format tables (MK83-5a/5b/6), by the same byte-provenance method used
for the Xerox-820-16-8 boot disk. It will be published here together with
the recipe when it boots the MAME `mk83` machine.
